Class: CrewMember

#CrewLeader#CrewMember(crewID, contactID, description, crewCategoryID, crewCategoryName, contactName, phone, email, isResponsible, contractSigned, hasBeenPaid)

Class used to store all the information for a crew member. It also contains information on if a given crew member is responsible for their given category. This necessitates that crewMember objects get created on a per event basis.

Constructor

new CrewMember(crewID, contactID, description, crewCategoryID, crewCategoryName, contactName, phone, email, isResponsible, contractSigned, hasBeenPaid)
