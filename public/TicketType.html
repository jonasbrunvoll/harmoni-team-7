Class: TicketType

TicketType(ticketTypeID, ticketTypenName, price, amount, releaseDate, releaseTime, hasEndDate, endDate, endTime, description)

Holds all the data for a ticket type. NOTE: there is not a separate object for each individual ticket.

Constructor

new TicketType(ticketTypeID, ticketTypenName, price, amount, releaseDate, releaseTime, hasEndDate, endDate, endTime, description)

Parameters:
Name Type Description
ticketTypeID int The ID of the ticket type. Generated by the database
ticketTypenName String Name of the ticket type.
price int The price of the ticket type.
amount int How many tickets exist of this ticket type.
releaseDate String The date when this ticket type gets released to the public. (yyyy-mm-dd)
releaseTime String The time when this ticket type gets released to the public. (hh:mm)
hasEndDate boolean Whether or not the ticket type should have an end date and end type.
endDate String The date when the ticket type no longer will be available. (yyyy-mm-dd)
endTime String The time when the ticket type no longer will be available. (hh:mm)
description String A description of the ticket type.
