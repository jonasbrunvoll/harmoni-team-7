Class: Event

Event(eventID, eventName, startDate, endDate, startTime, endTime, address, town, zipCode, status, description, publishDate, publishTime, organizer, eventType, picture)

The central class for the app. This class contains all the information for an event.

Constructor

new Event(eventID, eventName, startDate, endDate, startTime, endTime, address, town, zipCode, status, description, publishDate, publishTime, organizer, eventType, picture)

Parameters:
Name Type Description
eventID int The ID of the event. Generated by the database.
eventName String The name of the event.
startDate String The starting date of the event (yyyy-mm-dd)
endDate String The end date of the event (yyyy-mm-dd)
startTime String The starting time of the event (hh:mm)
endTime String The end time of the event (hh:mm)
address String The address for the event
town String The town where the event is being held
zipCode int The zipcode for the event addresss
status int The ID of the status of the event (0 = under planning, 1 = published, 2 = archived, 3 = canceled)
description String A description of the event
publishDate String The publishing date of the event (yyyy-mm-dd)
publishTime String The time of publishing for the event (hh:mm)
organizer Organizer The organizer object responsible for the event
eventType int The type of event, represented by an ID. The event types are static and fetched from the database during start-up and saved in EventStore.
picture PictureElement A picture element of the header picture for the event.
